# TheUltimateTest

## English
The Ultimate Test: A comprehensive survey to discover your inner self through MBTI, Zodiac, Numerology, and more. This interactive application provides detailed insights into your personality, elemental affinity, and even historical events associated with your birthdate.

### How it's made
This project was built with a lot of love and a touch of nostalgia. Here's what makes it tick:

*   **HTML5 & CSS3**: The backbone of the application. We used standard semantic HTML and handcrafted CSS (no frameworks!) to achieve that authentic retro CRT monitor look, complete with scanlines and pixel-perfect styling.
*   **Vanilla JavaScript (ES6+)**: The brain behind the magic. No heavy libraries here—just pure, efficient JavaScript handling the logic for MBTI calculations, Zodiac determination, and Numerology paths.
*   **Wikipedia API**: To make your results even more personal, we connect to the Wikipedia API to fetch a real historical event that happened on your birthday, localized to your preferred language.
*   **Google Fonts**: We used the "Press Start 2P" font to give everything that classic 8-bit arcade vibe.

## Español
El Test Definitivo: Una encuesta completa para descubrir tu yo interior a través de MBTI, Zodíaco, Numerología y más. Esta aplicación interactiva proporciona información detallada sobre tu personalidad, afinidad elemental e incluso eventos históricos asociados con tu fecha de nacimiento.

### Cómo está hecho
Este proyecto fue construido con mucho cariño y un toque de nostalgia. Aquí está lo que lo hace funcionar:

*   **HTML5 y CSS3**: La columna vertebral de la aplicación. Usamos HTML semántico estándar y CSS artesanal (¡sin frameworks!) para lograr ese auténtico aspecto de monitor CRT retro, completo con líneas de escaneo y estilo pixel-perfect.
*   **Vanilla JavaScript (ES6+)**: El cerebro detrás de la magia. Aquí no hay librerías pesadas, solo JavaScript puro y eficiente manejando la lógica para los cálculos de MBTI, la determinación del Zodíaco y los caminos de la Numerología.
*   **API de Wikipedia**: Para hacer tus resultados aún más personales, nos conectamos a la API de Wikipedia para buscar un evento histórico real que sucedió en tu cumpleaños, localizado a tu idioma preferido.
*   **Google Fonts**: Utilizamos la fuente "Press Start 2P" para darle a todo esa clásica vibra de arcade de 8 bits.
